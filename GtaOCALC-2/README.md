# Grind Planner

**Plan the grind. Hit the number.**

A no-nonsense calculator for crime-sim players. Pick your activities, set a money target, and get a day-by-day schedule that accounts for cooldowns, setup downtime, and passive-business accrual — no guesswork, no spreadsheets.

## Features

- **Chronological day simulator** with real cooldowns (Cayo won't reappear until it's actually ready)
- **Passive accrual tracking** for nightclub-style businesses — see pending $ at day's end
- **Deadline ↔ daily hours** — binary-searches the hours/day you need for any deadline
- **Fully editable** payouts, durations, cooldowns, and downtime — add your own activities
- **Local-first** — settings persist in `localStorage`, nothing sent to a server
- **Single self-contained HTML file** — no build step, no dependencies

## Usage

Open `index.html` in any modern browser, or visit the hosted version.

The landing page embeds the calculator inline. For a full-screen view, open `calculator.html` directly.
